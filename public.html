Fábrica de Software — Nosso Projeto

Este sistema foi desenvolvido como um projeto de conclusão do 2º ano da ETEC Heliópolis.

A equipe entrevistou a Rotisserie Israel e identificou problemas com papel, cardápio e pedidos.

A partir disso, criamos um sistema completo em PHP + MySQL e uma ferramenta de estoque em Python.

Acessibilidade Inclusiva

O sistema foi desenvolvido pensando em todos os públicos. Para isso, inclui recursos de acessibilidade como aumento e redução de fonte, dark mode, alto contraste, remoção de animações e leitura de itens.

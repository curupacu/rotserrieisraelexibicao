Sobre o Projeto — Fábrica de Software

Este sistema foi desenvolvido como parte da disciplina Fábrica de Software da ETEC, funcionando como um projeto de conclusão do segundo ano. Os grupos foram sorteados e cada equipe recebeu uma empresa real para desenvolver uma solução tecnológica.


Entrevistamos os responsáveis da Rotisserie Israel e identificamos problemas com o cardápio, organização de pedidos e falta de digitalização. A partir disso fizemos toda a análise, modelagem e criação do sistema web e da ferramenta em Python para controle de estoque.
